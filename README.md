# Getting Started

## Prerequisites

- npm

## Installation

1. Clone the repository

```bash
git clone https://github.com/blckfrost/clear-text
```

2. Navigate to project directory

```bash
cd clear-text
```

3. Install dependencies

```bash
npm install
```

## Usage

Run the project with:

```bash
npm run dev
```
